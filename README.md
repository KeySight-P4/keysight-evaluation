# Evaluation for KeySight

## Parameters of KeySight
```
-n or --netsight 	 Enable NetSight. 
-e or --everflow 	 Enable EverFlow. 
-k or --keysight 	 Enable KeySight. 
-m or --motivation 	 Enable KeySight. 
-M or --bf_size [BF_SIZE] 	 Set the array size of BF. 
-S or --window_num [WINDOW_NUM] 	 Set the window number of BF. 
-B or --bucket_num [BUCKET_NUM] 	 Set the bucket number of BF. 
-x or --bf_max [MAX of BF Cell] 	 Set the max cell size of BF. 
-N or --packet_per_window [Packet number] Set the number of packets per window. 
-R or --report_period [Period] 	 Set the reprt period. 
-a or --bf_alg [BF_ALG] 	 Set the algorithm of BF. 
    BSBF : 0
    BSBFSD : 1
    RLBSBF : 2
    SBF : 3
    BFQ : 4
    RBF : 5
```
